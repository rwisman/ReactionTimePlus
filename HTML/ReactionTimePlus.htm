Google Play - ReactionTime+

Description

ReactionTime+ measures reaction time from a prompt (visual and touch) until the user reacts by pressing a button attached to the phone or tablet headset jack.

ReactionTime+ uses an attached button for a more accurate and flexible measure than responding to an screen button tap.

Using a push button attached to the headset jack allows accurate time measurement based on audio recording at precise intervals (e.g. 44100 Hz). Using screen buttons to measure reaction time suffers from the variable latency of response to a screen tap, inherently less predictable or accurate.

Using an attached button for the reaction is adaptable for science education experiments that measure the user reaction time at the:

  • hand,
  • foot,
  • wherever pushing a button can be produced.

 

Press Start to begin a trial. Prompt for right foot reaction time. Stop reaction time trial with push button. List of reaction time results, left foot, etc.

The two signals initiating a single reaction time test and when the user should respond with a push button are:

  1. visual - the screen blinks red,
  2. touch - the phone vibrates (default is off).

The two signals allow random testing reaction times dependent upon two different senses. Any combination of hands or feet can be tested.

Use

  1. A response time trial is the duration from an visual or touch prompt until a response is detected; if not detected in 2 seconds no reaction time is measured.
  2. The reaction time trial can occur once or be repeated, results are displayed when measured and can be saved to the device.
  3. Multiple push buttons can be connected.
  4. Detection level can be adjusted via the menu if needed to accurately detect reaction time. If "Missing" occurs often after a valid reaction, increase the level.
  5. Reaction times are known to differ when responding to the touch or visual prompt.
  6. Response time trials have a default random delay of 1-4 seconds waiting for the trial and 2 seconds to respond. The delay can be changed to 1-10 seconds.
  7. The current limit of successfully timed trials is 1000.

Circuit and switch

The response mechanism includes a simple resistor/capacitor circuit given below and one or more switches. The circuit is connected through the headset jack.

Multiple switches can be connected in parallel to measure reactions at the hands/feet. Circuit A is simple but one switch cannot be distinguished from another. Circuit B is more complex and requires an initial calibration but allows each switch to be identified.

  • Momentary, normally open switches must be used.
  • Switches connected to the R3 or R4 resistor are normally open, effectively removing that resistor from the circuit.
  • Switches are connected in parallel, closing any switch places R3 or R4 in the circuit.
  • In circuit A, R3 value need not be precise, somewhat lower values should work equally well as should values below about 300Ω.
  • In circuit B, one switch is a direct connection while R3 and R4 should differ by 500Ω.
